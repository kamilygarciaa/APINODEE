##### PROJETOS:

###### comandos do terminal:

-ls : acessa tds os arquivos

\- cd : celeciona determinada pasta

\- npm: "Node Package Manager" permite aos desenvolvedores compartilhar e usar pacotes de código reutilizável, gerenciar as dependências do projeto (listadas no arquivo package.json) e automatizar tarefas.

\- cls: limpar

\- npm i express:  instala o pacote Express.js no seu projeto Node.js

\- npm init:  usado para inicializar um novo projeto JavaScript.





###### métodos http:

C - CREATE = POST: Enviar dados

R - READ = GET: Ler dados

U - UPDATE = PUT/PATCH: atualizar

D - DELETE = DELETE: deletar
